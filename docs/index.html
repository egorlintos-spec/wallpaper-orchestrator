Privacy Policy

Application: Wallpaper personal content (App ID 1585101) · Company: FallenOWL · Last updated: 2026-06-27

Wallpaper Orchestrator is a personal desktop application that publishes the owner's own original wallpaper and design images to the owner's own Pinterest business account, and retrieves analytics (impressions, saves, clicks) for those pins. It is a single-user personal tool and is not offered to or used by third-party users.

1. Data we access

Through the Pinterest API, the application accesses only the account of the authenticated owner, specifically:

  • Boards and pins owned by the account (to create new pins).
  • Pin and account analytics (impressions, saves, link clicks) for reporting.

2. Data we store

The application runs locally on the owner's computer. It stores only:

  • API access/refresh tokens, kept locally in a .env file on the owner's machine.
  • A local ledger of which images have been posted (posted.json).
  • Optional locally generated analytics reports (CSV/HTML).

No data is sent to any third-party server controlled by us. We do not operate any backend; all processing happens on the owner's device.

3. Data we do NOT collect

We do not collect, sell, or share any personal data of other Pinterest users. The application only interacts with the owner's own account.

4. Data sharing

No data is shared with third parties. Communication occurs solely between the owner's device and the official Pinterest API.

5. Data retention & deletion

All data is local. The owner may delete tokens, the ledger, and reports at any time by removing the corresponding files, or by revoking the app's access in their Pinterest account settings.

6. Security

Tokens are stored locally and never transmitted except to Pinterest over HTTPS.

7. Contact

For questions about this policy, contact the developer via the project repository: github.com/egorlintos-spec/wallpaper-orchestrator.

This application uses the Pinterest API but is not endorsed or certified by Pinterest.
